Proyecto de evaluación, donde pondremos a prueba los conocimientos del modulo 2.

Consiste en desarrollar una tienda online de productos con funcionalidad de búsqueda, carrito de la compra y almacenamiento en localStorage.

Esta aplicación permite:

- Buscar productos por su título.
- Añadir productos al carrito de la compra.
- Visualizar los productos añadidos en un listado.
- Mantener el carrito al recargar la página gracias al localStorage.

El proyecto está publicado en GitHub Pages, por lo que no es necesario abrir archivos localmente:

- Accede a la URL de GitHub Pages del proyecto:
  https://
- Usa el formulario de búsqueda para encontrar productos.
- Haz clic en Comprar para añadir productos al carrito.
- El carrito se actualizará y mostrará los productos añadidos

Tecnologías utilizadas:

- HTML5
- CSS3 / Sass
- JavaScript (ES6)
- LocalStorage
- API externa: [Fake Store API](https://fakestoreapi.com/products)
